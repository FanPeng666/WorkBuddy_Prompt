# WorkBuddy Prompt 自动化任务配置

WorkBuddy AI 自动化任务的 Prompt 配置中心，包含 B 站视频自动收藏和抖音自动续火花两个场景的完整任务定义。

## 项目结构

```
├── README.md                    # 本文件
├── B站高质视频收藏.md            # 硬核科普视频自动筛选收藏
├── B站有趣视频收藏.md            # 轻松有趣视频自动筛选收藏
├── 抖音自动续火花.md              # 好友火花自动续期
└── 抖音自动续火花出错/            # 故障诊断记录
    ├── 抖音续火花出错报告1.md
    └── 抖音续火花出错报告2.md
```

## 任务概述

### B 站视频收藏
- **高质视频**：自动化筛选科学/技术/人文领域的硬核科普视频，按专业度 + 数据指标综合评分，每日精选 3 个收藏到 `quality_video` 收藏夹
- **有趣视频**：发现创意、搞笑、有诚意的轻量内容，以"让人会心一笑"为目标

### 抖音自动续火花
- 自动检测有火花标识的好友，发送 🔥 文本消息维持火花连续性
- 支持云端 headless Chromium + Playwright 浏览器自动化
- 包含完整的异常处理、验证码检测和重试机制

## 技术栈

| 组件 | 用途 |
|---|---|
| **bilibili_api** (Python) | B 站视频搜索、信息获取、收藏操作 |
| **Playwright + Chromium** | 抖音 Web 端浏览器自动化（登录、消息发送） |
| **腾讯文档 MCP** | Prompt 配置托管与版本管理 |
| **WorkBuddy 自动化** | 定时调度、云端执行 |

## 使用方式

1. 将本仓库 Prompt 文档上传至腾讯文档
2. 在 WorkBuddy 中创建自动化任务，配置来源指向文档链接
3. 在自动化 prompt 中填入对应的 Cookie（B 站 SESSDATA / 抖音精简 8 字段）
4. 设置定时触发（建议每日一次）

## 安全约定

- ❌ Cookie 不写入文件、不写入日志、不写入代码注释
- ❌ 不使用外部未验证的依赖包
- ✅ 所有操作模拟真实用户行为
- ✅ 任务完成后清理临时数据
